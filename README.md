# Cybersecurity Awareness Chatbot

## Description
This is a cybersecurity awareness chatbot build with C# WinForms. It teaches users on cybersecurity topics such as password safety, phising, safe browsing, and two-factor authentication. The chatbot features sentiment detection, memory recall, random reponses and a GUI interface, task management, a quiz, NLP simulation, and an activity log.


## Features
- GUI interface with chat display
- ASCII art and welcome sound
- Keyword recongition for cybersecurity topics
- Random responses for varied interactions
- Sentiment detection based on user mood
- Memory and recall of user interest
- Error handling for unexpected inputs
- **Task Assistant** - Add, view, delete, and complete cybersecurity tasks with reminders
- **Cybersecurity Quiz** - Test your knowledge with 11 questions
- **NLP Simulation** - Recognizes different ways users phrase requests
- **Activity Log** - Tracks all actions with timestamps


## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Press F5 or click Run
4. Enter your name when
5. Start chatting about cybersecurity!

## Commands You Can Use
- `add task [task name]` - Add a new cybersecurity task
- `show tasks` - View your task list
- `start quiz` - Take the cybersecurity quiz
- `show log` - View recent activity log
- `what have you done` - View recent activity log
- `exit` - Close the application

## Topicss You Can Ask About
- Password safety
- Phishing
- Safe browsing
- Two-factor authentication (2fa)
- Scams
- Privacy
